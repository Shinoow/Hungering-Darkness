• Now uses DarknessLib
  • Removed the "Dynamic Lights Mode" and "Dynamic Lights List" config options (in favor of DarknessLib)
  • The timer pauses if you're riding in a minecart, boat, or similar vehicle
  • Dynamic Light is now added to the player's light level, instead of defaulting to bright enough if holding any handheld light source
  • Added a config option to go back to the old Dynamic Lights behavior
  • Added a configurable list of dimension-specific overrides for the majority of the available config options
  • Added a configurable biome whitelist for biomes where the darkness can hurt players
  • Added a config option to turn the biome whitelist into a blacklist
